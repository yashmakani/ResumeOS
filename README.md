# ResumeOS — Professional Resume Builder

ResumeOS is a fully client-side resume builder built with pure HTML, CSS, and JavaScript.
No frameworks, no backend, no installation required — just open and start building.

## ✨ Features

- 🎨 **23 Professional Templates** — Executive, Midnight, Crimson, Violet, Dusk and more
- 👁 **Live A4 Preview** — Real-time preview updates as you type
- 📄 **Perfect PDF Export** — Pixel-perfect A4 PDF matching your preview exactly
- 📸 **Passport Photo Upload** — Auto-positioned in every template header
- 🔗 **Clickable Contact Links** — Email, phone, LinkedIn links work in exported PDF
- 📱 **Fully Responsive** — Works on desktop, tablet, and mobile
- ✦ **AI Content Assistant** — Coming Soon
- 💾 **Import Existing Resume** — Coming Soon
- 🎯 **ATS Optimization** — Coming Soon

## 🗂 Files

| File | Description |
|------|-------------|
| `resumeos-landing.html` | Landing page — Hero, Features, Templates showcase |
| `resume-builder.html` | Full resume builder app |

## 🚀 Usage

1. Download both files into the same folder
2. Open `resumeos-landing.html` in any browser
3. Click **Start Building** → builder opens in new tab
4. Fill in your details, pick a template, export PDF

## 🛠 Built With

- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Playfair Display, DM Sans, DM Mono)
- No dependencies, no npm, no build tools
